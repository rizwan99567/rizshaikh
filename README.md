# Python Table Formatting

This project prints a table with centered text using string formatting in Python.

## Output
+------+----------+--------+
| roll | name | marks |
+------+----------+--------+
| 1 | Jay | 88 |
+------+----------+--------+
| 2 | Viru | 77 |
+------+----------+--------+

## Concepts Used
- Python string formatting
- Center alignment using `:^`
